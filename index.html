How It Works

The animatedCover.js extension replaces the static album image with a <video> element for selected tracks.
Works in both main view and full-screen (TV) mode.
You manage everything via a simple videoMap.

Installation

First Way:

  • Install 'Animated Covers' from Spicetify marketplace

Second Way:

  • Copy animatedCover.js to your Spicetify Extensions folder (e.g., C:\Users\AppData\Roaming\spicetify\Extensions)
  • Edit your config-xpui.ini and add this line under [Extensions]: animatedCover.js
  • Apply the changes: spicetify apply (in PowerShell)
